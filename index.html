Weather Station

I'm currently in the process of making a DIY project for a weather station. It's based on the NodeMCU board, which is capable of connecting to WiFi and sending HTTP requests containing data from sensors. I've already written API and database for this purpose, both hosted on AWS. Now I'm writing software for the board and the last step will be creating a beautiful dashboard website.

Technologies used: AWS (DynamoDB, API gateway, Lambda), C, HTML, Sass, ReactJS, Postman (API testing)
